method getMap [line 61]

void getMap( int $_size, BBox|Point $_arg0, [int|null $_arg1 = null], [array|null $_options = null])

Map getter; the main method in this class (except constructor). Possible forms are: getMap(BBox) and getMap(center, zoom)



Parameters:

int   $_size   Map tile size. Should be the power of 2.
BBox|Point   $_arg0   Either BBox or Point object.
int|null   $_arg1   "zoom" value if the first argument is Point object or null if not.
array|null   $_options   Optional additional arguments. Possible keys are:
  • format The image format of the map. Can be one of png for 8-bit PNG images, png32 for 32-bit PNG images, jpg for JPEG images or gif for GIF images. Defaults to png.
  • style Style id to be used for displaying map. Defaults to 1.
  • marker Sets markers on the map. Possible values are: Marker object, Marker objects array or array of data, needed to generate Marker object. In the last case, you can pass an array of arrays of data needed.
  • path Sets paths on the map image. Possible values are: Path object, Path objects array or array of data, needed to create a path. You can pass array of arrays of data, if needed.
