		find (line 50)
	
 
	
	

Find objects that match given query  Query should be in format [POI],[House Number],[Street],[City],[County]  like "Potsdamer Platz, Berlin, Germany".


Also supports "near" in queries, e.g. "hotel near Potsdamer Platz, Berlin, Germany"
