cm_get_tile (line 104)

A wrapper for getting tiles.

This is a thin wrapper aroung tile.Tile's get() method. If you don't want to pass a connection object, you can specify connection.Connection arguments after all other arguments. For example: get_tile(47.26117, 9.59882, 10, 1, 256, new Connection('BC9A493B41014CAABB98F0471D759707'));
